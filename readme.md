

设计一款手机可用的月度排班表 PWA 应用，兼具实用性和视觉设计空间。高颜值、可直接运行的实现方案。它采用现代卡片式设计，支持响应式布局（手机 / 电脑都好看），并包含完整的日期标记、 hover 详情和深色模式。
🎨 核心设计亮点
视觉分层：月份标题悬浮于日历上方，日期采用卡片化设计，增强立体感。手动点击切换月份。
状态色彩：预设了 白班 (蓝色)、夜班 (橙色)、休息 (紫色) 三种高辨识度配色。事假 / 病假 / 年假也使用不同颜色标注
交互反馈：日期 hover 时轻微上浮，点击可弹出快捷菜单修改状态。
深色模式：内置一键切换，适配夜间使用习惯。
完整实现代码 (HTML + CSS + JavaScript)
导出打印：添加一个按钮，将日历导出为 PDF 或直接打印。

直接在手机 PWA 界面上点某一天 → 选择班次 / 请假 → 保存，所有人立刻同步看到最新排班。

可在界面直接编辑的排班表 PWA + Gist 保存
完全基于你现有的架构，不用服务器、不用数据库，可以免费部署到 GitHub Pages。

页面加载 → 从 Gist 读取排班，gist中存放了一整年的排班数据
点击某一天 → 弹出选择框
白班 / 夜班 / 休息 / 事假 / 病假 / 年假
点保存 → 前端直接调用 GitHub Gist API 更新 JSON 格式如下
[
  {"date":"2026-04-01","type":"day","text":"白班"},
  {"date":"2026-04-02","type":"night","text":"夜班"},
  {"date":"2026-04-03","type":"rest","text":"休息"}
]

// ===== 宏：在此处写入你的 Gist ID 和 Token =====
DEFAULT_GIST_ID: 'your-gist-id-here',
DEFAULT_GIST_TOKEN: 'your-github-token-here',
// ================================================
使用方式：

创建 GitHub Gist，获取 Gist ID（如 a1b2c3d4e5f6...）
创建 GitHub Personal Access Token（勾选 gist 权限）
替换上面两个占位符的值
部署后所有人共享同一个 Gist，数据实时同步

所有人刷新页面就能看到最新排班
