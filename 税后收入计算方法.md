# 加拿大安大略省工资税后计算方法

## 一、工资构成

工资包（Gross Pay）包含：
- **基本工资** = 工时 × 时薪
- **晚班津贴**（Evening Premium, 15:00-23:00）
- **夜班津贴**（Night Shift Premium, 23:00-07:00）
- **周末津贴**（Weekend Premium）

## 二、CPP（Canada Pension Plan）2024

类似中国的养老保险，雇员和雇主各缴一半。

| 项目 | 值 |
|------|-----|
| 雇员缴费率 | 5.95% |
| 年最高可参保收入 | $68,500 |
| 基本豁免额 | $3,500 |
| 年最大CPP扣除 | $4,055.50 |

**计算公式：**
```
CPP扣除 = (年度参保收入 - $3,500) × 5.95%
```

> 年薪超过 $68,500 后，多出部分不扣 CPP。

## 三、EI（Employment Insurance）2024

类似中国失业保险。

| 项目 | 值 |
|------|-----|
| 雇员缴费率 | 1.66% |
| 年最大可参保收入 | $63,200 |
| 年最大EI扣除 | $1,049.12 |

> 年薪超过 $63,200 后，多出部分不扣 EI。

## 四、联邦税（Federal Tax）2024

**累进税率**，分段计算：

| 应税收入区间 | 税率 |
|---|---|
| $0 – $55,867 | 15% |
| $55,868 – $111,733 | 20.5% |
| $111,734 – $173,205 | 26% |
| $173,206 – $246,752 | 29% |
| $246,753+ | 33% |

## 五、安大略省税（Ontario Provincial Tax）2024

**累进税率**，分段计算：

| 应税收入区间 | 税率 |
|---|---|
| $0 – $51,782 | 5.05% |
| $51,783 – $103,564 | 9.15% |
| $103,565 – $150,000 | 11.16% |
| $150,001 – $220,000 | 12.16% |
| $220,001+ | 13.16% |

## 六、计算步骤

```
Step 1: Gross Pay = 基本工资 + 津贴总和
Step 2: 年度 Gross = 单期Gross × 26（双周次数）
Step 3: CPP年度 = (min(年度收入, $68,500) - $3,500) × 5.95%
         CPP单期 = CPP年度 / 26
Step 4: EI年度 = min(年度收入, $63,200) × 1.66%
         EI单期 = EI年度 / 26
Step 5: 应税收入 = 年度Gross - CPP年度 - EI年度
Step 6: 联邦税年度 = 累进计算
Step 7: 安省税年度 = 累进计算
Step 8: 税后Net = Gross - CPP - EI - 联邦税 - 安省税
```

## 七、累进税率分段计算示例（年薪 $85,000）

### CPP
| 项目 | 计算 | 金额 |
|------|------|------|
| 应参保收入 | min($85,000, $68,500) - $3,500 | $65,000 |
| 年CPP | $65,000 × 5.95% | $3,867.50 |
| 单期CPP | $3,867.50 / 26 | $148.75 |

### EI
| 项目 | 计算 | 金额 |
|------|------|------|
| 应参保收入 | min($85,000, $63,200) | $63,200 |

### 联邦税（应税收入 $80,083.38）
| 区间 | 计算 | 金额 |
|------|------|------|
| $55,867 × 15% | | $8,380.05 |
| ($80,083.38 - $55,867) × 20.5% | $24,216.38 × 20.5% | $4,964.28 |
| 合计 | | $13,344.33 |

| 区间 | 计算 | 金额 |
|------|------|------|

## 八、RRSP 说明

**RRSP（Registered Retirement Savings Plan）** = 注册退休储蓄计划

特点：
- 存进去的钱当年免税（减少应税收入）
- 账户内投资收益免税增长
- 取出时算当年收入、退休后通常税率较低
- 每年可存额度 = 年收入的 18%（2024年上限约 $31,560）

**不存RRSP：** 收入→交税→税后存钱→投资收益再交税
**存RRSP：** 收入→存RRSP（免税）→投资收益免税→退休取钱再交税

本质是"延迟交税"，利用现在税率高、退休后税率低的特点省税。

## 九、实现说明

实现此功能时：
1. 使用安大略省2024税率
2. CPP豁免额 $3,500/年
3. CPP可参保收入上限 $68,500，EI上限 $63,200
4. 按26个双周分摊年度扣除
5. 暂不考虑RRSP、自雇税率等复杂情况
6. 税率每年需要更新配置
